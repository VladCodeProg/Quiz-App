# Quiz app created with React
---------------------
# How to use?
1. Clone respository
2. Run this commands:
*  'npm i'
*  'npm i axios sass react-router-dom react-transition-group'
4. Also you need firebase realtime database with branch named 'quizes'
5. In quizes create quizItem by this example: 
  answers: [
      0: id: ...  text: ... 
      1: id: ...  text: ... 
      2: id: ...  text: ... 
      3: id: ...  text: ... 
  ]
  id: 1
  question: "..."
  rightAnswerId: 1
---
# Enjoy)
